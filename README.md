# 週間タイムブロッカー

スマートフォンで使いやすい、1週間分の予定を時間帯ごとに整理できるシンプルなタイムブロッカーです。
予定は15分刻みで開始・終了時刻を調整でき、優先度ごとに色分けして表示できます。
スマートフォンでは横スクロールで各曜日を広く表示し、予定の文字を読みやすくしています。

予定はブラウザのローカルストレージに保存され、JSONバックアップの書き出し・読み込みにも対応しています。

## 携帯でオフライン利用する方法

GitHub Pages などの HTTPS で公開された URL をスマートフォンで一度開くと、アプリ本体が端末にキャッシュされます。
その後、ブラウザの「ホーム画面に追加」から追加すると、通信がない場所でも起動して予定を確認・編集できます。

予定データは端末内のブラウザ保存領域に残ります。機種変更やブラウザ削除に備えて、バックアップ画面から定期的に JSON を書き出してください。
